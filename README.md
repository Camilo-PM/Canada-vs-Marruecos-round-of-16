# Predicción Octavos de Final - Mundial 2026 ⚽📊

Proyecto de analítica predictiva utilizando Machine Learning para simular las llaves de octavos de final de la Copa del Mundo, analizando el histórico desde 1930.

## Tecnologías utilizadas
* **Python** (Pandas, Scikit-Learn)
* **VS Code** & **Git/GitHub**
* **Tableau**
